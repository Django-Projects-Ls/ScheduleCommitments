# ScheduleCommitments

ScheduleCommitments is a web aplication for save and record your compromisses

## Setup Project

> On Windows:

```bash
py -m venv .venv
.venv/Scripts/activate
pip install -r requirements.txt
```

> On Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
